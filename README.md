A desktop-based Radar Cross Section (RCS) and Radar Cross Section Reduction (RCSR) analysis tool developed using Python and PyQt5. The application automates correction factor generation using calibrated sphere references, calculates corrected RCS values, and compares reference and metasurface measurements to determine RCSR performance.
RCS calculation using calibrated sphere reference measurements
Support for 4-inch, 8-inch, and 12-inch sphere references
Automatic correction factor generation
Corrected RCS computation from measurement data
RCSR calculation between reference and metasurface targets
Frequency vs RCS plotting
Frequency vs RCSR plotting
Multiple plot management
Interactive graph selection and removal
Data Viewer for correction factors and processed results
Automatic file format detection (.csv, .txt, .dat)
Export processed results to CSV
Dark/Light theme support
Real-time processing status updates
